# Monty Hall Problem (N Doors)

This project looks at the Monty Hall problem, but instead of only 3 doors, it extends the idea to any number of doors.

The main goal is to see how the probabilities change when the number of doors increases and to compare different strategies:
- staying with the initial choice
- switching to another door
- making a random decision

The project combines a simple theoretical explanation with simulations in Python. By running the simulation multiple times, we can estimate the probability of winning for each strategy.

The results show that switching gives a clear advantage, and this advantage becomes even stronger when more doors are added.

The code is written in Python and can be used to experiment with different numbers of doors and trials.
